# AGENTS.md

This file provides Codex-facing behavioral guidance for working in this repository.

## Principles

### 1) Think Before Coding
- Surface assumptions explicitly.
- Clarify ambiguity before implementation.
- State key tradeoffs when relevant.

### 2) Simplicity First
- Prefer the minimum solution that satisfies the request.
- Avoid speculative abstractions and unrequested flexibility.

### 3) Surgical Changes
- Modify only what is needed for the task.
- Match existing project style.
- Avoid unrelated refactors/formatting churn.

### 4) Goal-Driven Execution
- Define concrete success criteria.
- Break work into verifiable steps.
- Validate results before finalizing.
